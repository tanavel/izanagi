## Getting Started
### Prerequisites
- terraform
  ```
  brew install terraform
  ```

### Installation
- Clone the repo
  ```
  git clone git@github.com:tanavel/izanagi.git
  ```

## Usage
- Execute terraform
  ```
  cd {target_component_dir}
  terraform apply
  ```
